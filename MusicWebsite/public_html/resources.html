Borrowed HTML, CSS and Media Sources

I took the idea of a slideshow here: https://www.w3schools.com/howto/howto_js_slideshow.asp and I converted it
into a set of slides used to navigate through the genres, because I didn't want the user to leave from one site to another.

About the Site

  • a) Graphic created from scratch: All the drawings of the instruments on the different music genres were drawn
    from scratch by me.
  • b) I used embedded media from YouTube. These are included on the bottom of each page (excluding the home page
    that the user first lands on).
  • c) All images use the CSS image rollover effect, where after being hovered over by the user, they scale.
  • d) JavaScript is used for the functionality of the slide show (this was selected for easy animation on the page),
    JavaScript is also used for the background color to change for each slide. I also used JavaScript for a button to appear that the user
    can use to go to the top of the page.

Summary of the Site

I am very proud of all the JavaScript I used, as it was my first time implementing JavaScript and most of it was
based on my ideas and thoughts. I am also pleased with the way the pictures of the instruments that I drew turned out. Only recently did I
start using Illustrator and I am proud of how much I have learnt so far.
Another thing I feel was done really well was the consistency of the design. Every page seems in a way unique, yet they are all following
the same principle.
One factor I would've liked to have improved was the audio. To achieve what I wanted to do with the music was too complicated to be done. I
would've liked to have a different style and a different layout when on devices where the height is longer than the width.
All in all I am proud of what I managed to do after one course of Web and Mobile.
